# Call-centre-performance-dashboard
Power BI dashboard analyzing call centre KPIs such as answer rate, resolution rate, agent performance, and customer satisfaction.
![Dashboard](dashboard_screenshot.png)

## Key Insights
• The majority of incoming calls were successfully answered, indicating good call centre coverage.

• Resolution rate was slightly lower than the answer rate, suggesting that some customer issues require follow-up or escalation.

• Agent performance varied across total calls handled and average speed of answer.

• Customer satisfaction remained moderate, indicating potential opportunities to improve service quality and response efficiency.
