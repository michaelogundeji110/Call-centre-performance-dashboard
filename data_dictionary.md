# Data Dictionary

| Column Name | Description |
|-------------|-------------|
| Call ID | Unique identifier for each call |
| Date | Date the call occurred |
| Agent | Customer service agent handling the call |
| Topic | Category of the call (billing, support, etc.) |
| Answered | Indicates whether the call was answered (Y/N) |
| Resolved | Indicates whether the issue was resolved (Y/N) |
| Satisfaction | Customer satisfaction rating (1–5) |
| Speed of Answer (s) | Time taken before the call was answered |
